# 高校实习管理系统 java477

## 项目概述

高校实习管理系统是一个基于Springboot+Vue的前后端分离的Javaee项目，运用MVC三层架构设计，实现了实习流程的数字化管理。系统通过结合Spring、SpringBoot、Mybatis、Vue和Maven等技术开发，以MySQL作为数据库支持，打造了一套操作便捷、界面友好的B/S结构Web应用。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 构建工具：Maven
- 其他：Spring、Mybatis

## 功能模块

- 学生模块：实习申请、进度查询等
- 教师模块：实习分配、学生管理、成绩评定等
- 管理员模块：系统设置、权限管理、数据统计等

## 系统角色

- 学生
- 教师
- 管理员

## 运行环境

- 服务器：推荐使用JavaEE兼容的Web服务器，如Tomcat
- 数据库：MySQL（版本建议5.7及以上）
- 前端：现代浏览器（推荐Chrome、Firefox等）
- 后端：JDK1.8及以上版本

## 部署步骤

1. 准备JavaEE运行环境
2. 安装MySQL数据库，并导入初始数据
3. 部署后端Springboot应用
4. 部署前端Vue应用

## 目录结构

```
|-- 前端代码目录
|-- 后端代码目录
    |-- src
    |-- pom.xml
    |-- ...
|-- 数据库脚本目录
    |-- init.sql
|-- ...
```

## 核心亮点

- 前后端分离架构，易于维护和扩展
- 基于Springboot开发，保证了系统的稳定性和高效性
- MVC分层设计，代码结构清晰，易于理解和二次开发
